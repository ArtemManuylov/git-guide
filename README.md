# Руководство по Git и GitHub 

Цель данного документа: предоставить четкую, пошаговую инструкцию для выполнения базовых операций системы контроля версий Git через интерфейс командной строки Git Bash с последующей публикацией кода на хостинге GitHub.

## О проекте

Данный проект демонстрирует практическое применение современных подходов к созданию технической документации.

## Демонстрируемые навыки, инструменты и технологии

* **Topic-Based Writing** - модульный подход к структурированию документации;
* **Техническое письмо** - адаптация сложных тем для начинающей аудитории;
* **Информационная архитектура** - проектирование структуры документации;
* **Git** - система контроля версий;
* **GitHub** - хостинг проектов и совместная разработка;
* **GitHub Pages** - хостинг статической документации;
* **Docsify** - генератор статической документации;
* **Markdown** - форматирование технического контента;
* **Figma** - создание и редактирование скриншотов, визуальное оформление;
* **HTML/CSS** - кастомизация внешнего вида документации.
