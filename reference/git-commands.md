## Основные команды Git: краткий справочник

### Навигация

`pwd` — покажи текущую директорию;

`ls` — покажи файлы в текущей директории;

`cd ~` — перейди в домашнюю директорию;

`cd ..` — перейди на уровень выше.

### Работа с файлами и папками

`mkdir <имя>` — создай папку;

`touch <имя>` — создай файл;

`cat <имя>` — покажи содержимое файла.

### Инициализация репозитория

`git init` — инициализируй репозиторий;

`git clone <url>` — склонируй удалённый репозиторий.

### Синхронизация локального и удалённого репозиториев

`git remote add origin <url>` — привяжи удалённый репозиторий;

`git push -u origin main` — отправь изменения в удалённый репозиторий;

`git pull` — скачай изменения из удалённого репозитория.

### Подготовка файла к коммиту

`git add <имя>` — добавь конкретный файл;

`git add .` — добавь все файлы в текущей директории;

`git add --all` — добавь все изменённые файлы.

### Создание и публикация коммита

`git commit -m "текст"` — создай коммит с описанием;

`git push` — опубликуй коммиты в удалённом репозитории.

### Просмотр информации о коммитах

`git log` — покажи историю коммитов;

`git log --oneline` — покажи краткую историю коммитов;

`git show <хэш>` — покажи информацию о конкретном коммите.

### Просмотр состояния файлов

`git status` — покажи текущее состояние репозитория;

`git diff` — покажи изменения в файлах;

`git diff --staged` — покажи изменения в подготовленных файлах.

### Добавление изменений в последний коммит

`git commit --amend -m "новый текст"` — измени последний коммит.

### Восстановление файлов и отмена коммитов

`git restore <файл>` — откати файл к последнему коммиту;

`git restore --staged <файл>` — убери файл из подготовленных;

`git reset --hard <хэш>` — откатись к конкретному коммиту.

### Просмотр изменений

`git diff` — покажи неподготовленные изменения;

`git diff --staged` — покажи подготовленные изменения;

`git diff <хэш1> <хэш2>` — сравни два коммита.
