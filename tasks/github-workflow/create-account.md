## Регистрация на GitHub

Перед началом работы с Git необходимо создать учетную запись на GitHub. 

Выполните следующие шаги, чтобы создать свой профиль:

1. **Перейдите на сайт GitHub.**
   
    Откройте свой браузер и перейдите на [официальный сайт GitHub](https://github.com/).

2. **Заполните основную информацию.**
   
    На главной странице в правом верхнем углу нажмите на кнопку `Sign up`.

    ![Sing up](../../images/SignUp.png)

3. **Форма ввода данных.**

    Сайт перенаправит Вас на поле заполнения данных.

   * **Email** —  действующая почта.
   * **Password** —  надежный пароль.
   * **Username** —  уникальное имя для профиля.
   
   ![Data Entery](../../images/DataEntery.png)

   Нажмите `Create account`.
   
4. **Верификация.**
  
   На этом этапе GitHub проверяет, что регистрацию выполняет реальный человек, а не бот. Вам будет предложена простая интерактивная задача. Чаще всего это: *"Выберите все квадраты с объектами"*. Кликайте на все подходящие области изображения. После успешного решения нажмите `Continue`.

5. **Подтверждение email-адреса.**
  
    * Откройте почтовый ящик, указанный при регистрации.
    * В папке "Входящие" найдите письмо от [GitHub](https://github.com/).
    * Письмо содержит кнопку `Verify email address`. Просто нажмите на нее для подтверждения, или будет указан цифровой код, который необходимо ввести на странице GitHub.
  
Регистрация на GitHub завершена. Теперь вы можете перейти к созданию первого репозитория и настройке профиля.
