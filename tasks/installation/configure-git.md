## Настройки Git 

Перед началом работы необходимо указать своё имя и почту. Эта информация будет использоваться для подписи ваших коммитов. При этом не имеет значения, в какой директории вы находитесь, вызов `git config --global` сработает везде.

```bash
git config --global user.name "Ваше Имя"
git config --global user.email "username@mail.ru"
```
*Примечание: Замените "Ваше Имя" и "username@mail.ru" на данные, которые вы используете на GitHub. Имя необходимо написать латиницей.*

Для проверки настроек выполните: `git config --list`.

Если вы сделали всё верно, командная строка покажет текущее значение.
```bash
user.name "Ваше Имя"
user.email "username@mail.ru"
```
