## Просмотр изменений в файлах

Позволяет увидеть, что именно изменилось в файлах. Проверить код перед сохранением и найти ошибки перед коммитом.

* Посмотреть неподготовленные изменения:

```bash
# Изменения в файлах, которые ещё не добавлены через git add
git diff
```

* Посмотреть подготовленные изменения:

```bash
# Изменения в файлах, которые уже добавлены через git add
git diff --staged
```

* Сравнение конкретного файла:

```bash
# Только для README.md
git diff README.md

# Только подготовленные изменения для index.html
git diff --staged index.html
```
